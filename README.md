# Mina zkApp: Learning

This template uses TypeScript.

## How to build

```sh
npm run build
```

## how to setup
install zkapp-cli

```sh
zk config
zk deploy
```

## How to run

```sh
pm run build && node build/src/newinteract.js
pm run build && node build/src/newinteract2.js
```
